Nível 1: Criar um nó simples

Seu objetivo é criar um nó que armazene um valor e tenha uma referência para o próximo nó.

Instruções: Para criar um nó, você precisa usar um objeto com duas propriedades principais:

  • value: Esta propriedade armazenará o valor do nó.
  • next: Esta propriedade apontará para o próximo nó na lista ou será null se não houver próximo nó.

Exemplo de código:

let node = { value: 1, next: null };
